# Phase 5D 真实数据源字段映射函数草案

## 1. 本阶段说明

本阶段只创建字段映射函数草案。

本阶段没有接入真实 API。

本阶段没有调用外部网络。

本阶段没有读取真实行情数据。

本阶段没有生成真实行情数据。

## 2. 映射函数职责

映射函数只负责把未来外部数据转换成系统标准字段。

映射函数不得用假数据补齐缺失字段。

缺失字段必须保留 `None`，后续显示为“数据不足”。

映射后的数据默认 `data_status=待人工确认`。

映射后的数据不能自动进入“已验证”。

人工确认前不得用于真实市场环境结论。

## 3. 安全边界

* 市场环境结论不是交易指令。
* 本系统不自动交易，不自动下单。
* 本模块不做个股推荐。
* 映射函数不输出投资建议。
* 映射函数不输出交易建议。
* 映射函数不生成目标价。

## 4. 当前结论

`core/market_data_mapper.py` 是字段映射草案。

`tests/test_market_data_mapper.py` 只使用 mock 输入验证字段结构。

当前阶段仍未选择真实数据源，未配置真实 endpoint，未读取真实行情。
