# Phase 5C 真实数据源配置草案

## 1. 本阶段说明

本阶段只创建真实数据源配置草案。

当前没有选择最终真实数据源。

当前没有接入 API。

当前没有调用外部网络。

当前没有真实行情数据。

## 2. 配置状态

* 所有候选源 `enabled=false`。
* 所有 `endpoint_configured=false`。
* `selected_source=null`。
* API Key 未来只能通过环境变量读取。
* 当前不得写入真实 API Key。
* 当前不得写入真实 token。
* 当前不得写入真实 endpoint。

## 3. 数据状态规则

* 真实数据首次接入后必须先进入“待人工确认”。
* 人工确认前不得标记为“已验证”。
* 字段缺失时必须显示“数据不足”。
* 数据异常时必须显示“数据不足”。
* 不得用假数据补齐缺失字段。

## 4. 字段边界

候选配置只能使用 `docs/17_MARKET_ENVIRONMENT_DATA_SCHEMA.md` 中定义的市场资金字段和市场情绪字段。

市场资金字段只允许围绕当前价、20日均线、50日均线、均线方向、成交量、相对成交量、是否强于基准、风险偏好扩散、数据来源和数据状态进行映射。

市场情绪字段只允许围绕指标 ID、指标名称、指标类型、当前值、当前状态、风险含义、变化方向、数据来源和数据状态进行映射。

## 5. 安全边界

* 行情数据不产生投资建议。
* 行情数据不产生交易建议。
* 行情数据不产生目标价。
* 市场环境结论不是交易指令。
* 本系统不自动交易，不自动下单。
* 本模块不做个股推荐。

## 6. 当前结论

`config/real_data_sources.draft.json` 只是配置草案。

该文件不代表已经选择真实数据源。

该文件不代表已经接入真实数据源。

该文件不包含真实行情数据。

后续进入真实接入前，必须先完成安全检查、字段映射检查、错误处理检查和人工确认流程。
