# Phase 5E-2A 真实数据启用开关与运行边界

## 1. 本阶段说明

本阶段只创建运行时安全开关。

本阶段没有接入真实 API。

本阶段没有调用外部网络。

本阶段没有真实行情数据。

## 2. 默认运行边界

* 默认状态禁止外部网络。
* 默认状态禁止真实行情进入 UI。
* 默认状态禁止自动标记“已验证”。
* 默认状态禁止投资建议。
* 默认状态禁止自动交易。
* 默认状态禁止自动下单。

## 3. 数据状态规则

真实数据首次接入后必须先标记为“待人工确认”。

字段缺失必须显示“数据不足”。

人工确认前不得标记为“已验证”。

不得用假数据补齐缺失字段。

## 4. 安全边界

市场环境结论不是交易指令。

本系统不自动交易，不自动下单。

本模块不做个股推荐。

运行时安全开关不能绕过真实数据源配置草案中的禁用状态。

## 5. 当前结论

`config/runtime_flags.example.json` 是运行时安全开关示例。

所有 `allow_*` 开关默认都是 `false`。

`core/runtime_flags.py` 只负责读取和校验本地运行时开关，不接入外部服务。
