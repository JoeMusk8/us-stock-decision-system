# Phase 5E-1 真实数据源只读接入骨架

## 1. 本阶段说明

本阶段只创建真实数据源只读接入骨架。

本阶段没有真正接入 API。

本阶段没有调用外部网络。

本阶段没有拉取真实行情。

所有候选源仍然 `enabled=false`。

`selected_source` 仍然为 `null`。

当前 `fetch_real_market_data_readonly()` 必须返回 `ok=false`。

当前阶段禁止外部网络调用。

## 2. 启用前规则

真实数据源未来启用前必须经过人工确认。

真实数据首次接入后必须先标记为“待人工确认”。

人工确认前不得标记为“已验证”。

不得写入真实 API Key。

不得写入真实 token。

不得写入真实 endpoint。

## 3. 安全边界

* 市场环境结论不是交易指令。
* 本系统不自动交易，不自动下单。
* 本模块不做个股推荐。
* 只读客户端骨架不得输出投资建议。
* 只读客户端骨架不得输出交易建议。
* 只读客户端骨架不得生成真实行情数据。

## 4. 当前结论

`core/real_data_source_client.py` 只是只读接入骨架。

当前版本只读取本地配置草案，并检查真实数据源是否仍处于关闭状态。

当前版本不会访问外部服务，不会返回真实行情 records，不会生成真实市场环境结论。
