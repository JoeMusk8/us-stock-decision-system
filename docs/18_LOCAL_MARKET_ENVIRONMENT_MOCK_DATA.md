# Phase 4C 本地市场环境 mock 数据

## 1. 说明

* 本阶段只创建本地 mock 数据结构。
* 当前没有接入真实 API。
* 当前没有真实行情数据。
* 当前没有真实宏观数据。
* mock 数据不能用于投资判断。
* mock 数据不能进入真实市场结论。
* 市场环境结论不是交易指令。
* 本系统不自动交易，不自动下单。
* 市场资金字段只包含 20日均线、50日均线。
* 禁止使用 20日与50日的涨跌幅。

## 2. mock 文件

本阶段新增文件：

* `data/market_environment_mock.example.json`

该文件只用于结构验证和后续本地读取测试，不代表任何真实行情、真实宏观数据或真实市场情绪。

## 3. 字段边界

市场资金 mock 数据覆盖：

* QQQ
* 纳斯达克指数
* 比特币

市场情绪 mock 数据覆盖：

* VIX

所有数值字段使用 `null`，不得用真实价格、真实成交量或真实指标值填充。

## 4. 安全边界

* mock 数据不是交易信号。
* mock 数据不产生买入建议。
* mock 数据不产生卖出建议。
* mock 数据不产生目标价。
* mock 数据不触发自动交易。
* mock 数据不触发自动下单。
