# Phase 4E 市场环境 UI 本地 mock 数据联通

## 1. 说明

* 本阶段只把 UI 连接到本地 mock 数据。
* 本阶段没有接入真实 API。
* 本阶段没有真实行情数据。
* 本阶段没有真实宏观数据。
* `market_funds` 和 `market_sentiment` 来自 `data/market_environment_mock.example.json`。
* mock 数据 `data_status=示例数据`，不能进入真实市场结论。
* 页面必须显示“数据不足”或“示例数据”。
* 市场环境结论不是交易指令。
* 本系统不自动交易，不自动下单。
* 下一阶段才允许做 mock UI 验收。

## 2. UI 联通范围

本阶段只联通市场资金 / 情绪 / 宏观监控模块中的本地 mock 数据展示。

联通页面：

* 市场环境监控主页面
* 市场资金监控
* 市场情绪监控
* 市场环境结论

## 3. 安全边界

* 不接 API。
* 不调用外部网络。
* 不生成真实行情数据。
* 不生成真实宏观数据。
* 不使用真实价格。
* 不使用真实成交量。
* 不输出投资建议。
* 不输出交易建议。
* 不自动交易。
* 不自动下单。
