# 美股交易决策系统

## 当前状态

* 本项目是 Streamlit Web App
* 当前入口文件是 app.py
* 当前已完成五个模块 UI
* 当前已完成市场环境模块 yfinance 读取逻辑
* 当前已完成 Streamlit Cloud 部署
* 当前已完成主链路会话级 MVP
* 当前已支持工作区 JSON 备份与恢复
* 当前已支持 SEC 公司事实最小查询

## 运行方式

本地运行：

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 部署

部署入口文件：

```text
app.py
```

Python 依赖文件：

```text
requirements.txt
```

## 模块

1. 首页总览
2. 范式创新监控
3. 基础投研 / 产业链图谱 / 候选股雷达
4. 深度投研
5. 重点跟踪池 / 量化策略标注
6. 市场资金 / 情绪 / 宏观监控

## 数据说明

* 市场环境模块优先读取 QQQ、纳斯达克指数、比特币、VIX
* 首页总览可按 ticker 读取 SEC submissions / companyfacts
* 数据状态默认待人工确认或数据不足
* 当前工作流数据为会话级数据，可通过 JSON 备份与恢复

## 工作区备份

当前支持工作区 JSON 备份与恢复，可保存并恢复：

* 范式主题
* 基础投研行业 / 主题池
* 候选股
* 深度投研任务
* 重点跟踪池
* SEC 公司事实快照

入口位于侧边栏“工作区备份”。

## SEC 公司事实查询

首页总览当前支持：

* 手动输入 ticker
* 读取 SEC CIK
* 读取最近 filings
* 读取 companyfacts 中的核心财务字段
* 结果保存为会话级 SEC 公司事实快照
* 数据状态为待人工确认或数据不足

## 市场环境模块能力

市场环境模块已支持：

* QQQ / ^IXIC / BTC-USD / ^VIX
* 20日均线 / 50日均线
* VIX 情绪状态
* 支持 / 谨慎 / 不支持 / 数据不足 的市场环境结论
* 该结论用于市场环境记录

## 重点跟踪池能力

重点跟踪池当前支持：

* 手动添加股票
* 股票 + 研究主题绑定
* 研究等级 S/A/B/C/D
* 跟踪状态
* 持仓状态
* 手动买入价 / 止盈价 / 止损价标注
* 策略结构风险标签
* CSV 导出
* 当前版本为会话级数据

## 深度投研模块能力

深度投研模块当前支持：

* 手动创建投研任务
* 股票 + 研究主题绑定
* 八项验证分数
* Claim / Evidence / Counter Evidence 记录
* 自动计算研究总分
* 自动生成研究等级 S/A/B/C/D
* 自动生成研究流转 decision
* CSV 导出
* 当前版本为会话级数据
* 研究等级需人工确认

## 模块联动能力

模块联动当前支持：

* 范式主题可转入基础投研行业/主题池
* 候选股可转入深度投研任务池
* 深度投研任务生成研究等级后，可手动转入重点跟踪池
* 转入时保留 ticker、theme、research_grade、decision、evidence_note、risk
* 如果同 ticker + theme 已存在，则更新，不重复追加
* 该流转是研究流程管理
* 不自动生成买入价、止盈价、止损价
* 手动确认后进入下一模块

## 基础投研模块能力

基础投研模块当前支持：

* 最多 3 个行业 / 主题输入
* 手动添加候选股
* 候选股转入深度投研
* CSV 导出
* 当前版本为会话级数据
* 候选股需人工确认

## 范式创新监控能力

范式创新监控当前支持：

* 手动添加新方向 / 新主题
* 范式主题转入基础投研行业/主题池
* 最多保留 3 个基础投研行业/主题
* 当前版本为会话级数据
* 范式主题需人工确认
