# UI 规约

本文件是项目 UI 页面结构的唯一事实源。后续实现 UI 前必须读取本文件。

## 全局 UI 风格

整体风格参考 Gemini：

* 浅灰背景
* 白色卡片
* 圆角
* 轻阴影
* 大面积留白
* 淡蓝 / 淡紫点缀
* 中文界面
* 桌面端 Web App

禁止：

* 不使用原始示意图里的大面积蓝色、黄色、绿色色块
* 不使用复杂花哨图表
* 不生成投资建议文案
* 不生成自动交易按钮

## 全局布局

左侧 sidebar 导航 5 个模块：

1. 范式创新监控
2. 基础投研 / 产业链图谱 / 候选股雷达
3. 深度投研
4. 重点跟踪池 / 量化策略标注
5. 市场环境监控

每个页面顶部必须显示：

* 模块标题
* 模块说明
* 数据状态：示例数据 / 待配置 / 数据不足
* 提示：本系统不提供投资建议，不自动交易

## 1. 范式创新监控模块 UI

共 7 页：

1. 24小时信息流总览
2. 美国政府资金流向
3. 巨头公司科研方向配置
4. 商界领袖关注方向配置
5. X KOL 发言配置
6. AI 审核队列
7. 24小时输出详情

必须包含：

* 信息流表格
* AI标签
* 来源可信度
* 人工审核状态
* 美国政府资金流向
* 巨头公司科研方向
* 商界领袖关注方向
* X KOL 发言

重要提示：
KOL 信息只能作为线索，不能直接作为事实。

## 2. 基础投研 / 产业链图谱 / 候选股雷达模块 UI

共 6 页：

1. 基础投研主工作台
2. 产业链全景图谱
3. 节点评分 / 瓶颈识别
4. 候选股雷达
5. 证据与 AI 解释
6. 人工确认 / 状态管理

必须包含：

* 最多 3 个行业输入框
* 6 层产业链图谱
* 10 个瓶颈评分维度
* 最多 5 只候选股票
* 证据ID
* AI解释草稿
* 人工确认状态

重要提示：
候选股雷达不是买入信号，只代表进入深度投研候选。

## 3. 深度投研模块 UI

共 11 页：

1. 股票任务池总览
2. 单股投研工作台总览
3. 公司业务边界验证
4. 主题受益验证
5. 产业链位置验证
6. 财务兑现验证
7. 客户 / 订单 / CapEx 验证
8. 竞争格局验证
9. 风险与反证验证
10. Claim 验证
11. 研究报告与等级

必须包含：

* 最多 15 只手动股票
* 股票 + 主题绑定
* 证据数量
* Claim 数量
* evidence_id
* 研究等级 S/A/B/C/D

重要提示：
研究等级不是买入评级，不代表交易建议。

## 4. 重点跟踪池 / 量化策略标注模块 UI

共 10 页：

1. 重点跟踪池主页面
2. 主题变量
3. 公司变量
4. 财务变量
5. 催化剂变量
6. 风险变量
7. 数据新鲜度
8. 量化策略标注详情
9. 新证据 / 新风险提醒
10. 复查任务

必须包含：

* 高质量标的列表
* 核心变量跟踪
* 新证据提醒
* 新风险提醒
* 数据新鲜度
* 复查任务
* 买入价格
* 止盈价格
* 止损价格

重要提示：
买入价格、止盈价格、止损价格只是策略标注字段，不是交易指令。
本项目不自动交易，不自动下单。
不单独生成量化交易模块。

## 5. 市场环境监控模块 UI

共 6 页：

1. 市场环境监控主页面
2. 市场资金监控
3. 市场情绪监控
4. 宏观环境监控
5. 市场环境结论
6. 时间提醒 / 事件日历

市场资金监控必须包含：

* QQQ
* 纳斯达克指数
* 比特币
* 当前价
* 20日均线
* 50日均线
* 20日均线方向
* 50日均线方向
* 成交量
* 相对成交量
* 是否强于基准
* 风险偏好扩散

禁止出现：

* 20日涨跌幅
* 50日涨跌幅

市场情绪监控必须包含：

* VIX
* Put / Call
* X 社交情绪

宏观环境监控必须包含：

* 核心利率
* 核心通胀
* 核心就业
* 核心GDP
* 原油

重要提示：
市场环境结论不是交易指令，只是辅助风控判断。
